# HTML Quest -  🏰

HTML Quest é um **mini-jogo educativo estilo videogame** para aprender HTML de forma divertida e interativa. O jogador avança por níveis resolvendo desafios de HTML, enquanto coleta moedas 💰 e evita obstáculos 🔴 no mapa. Cada nível tem dicas e feedback instantâneo.

---

## 🎮 Recursos

- **10 níveis educativos** do básico ao avançado.
- **Editor + Preview** integrados para testar HTML em tempo real.
- **Mapa interativo** com personagem animado 🧙‍♂️.
- **Moedas e obstáculos**, com pontuação.
- **Feedback instantâneo** e dicas visuais.
- **Efeitos sonoros** para moedas coletadas e vitória final.
- Fácil de expandir com novos níveis ou desafios.

---

## 🕹️ Como Jogar

1. Abra o arquivo `index.html` em qualquer navegador moderno.
2. Leia a descrição do nível na parte superior. 
3. Escreva o HTML correto no editor à direita.
4. Veja o resultado em tempo real no preview abaixo do editor.
5. Ao completar o nível, o personagem se move no mapa e você ganha moedas.
6. Colete moedas e avance para o próximo nível.
7. Complete todos os 10 níveis para vencer o jogo! 🎉

---

## 🧩 Níveis

1. Crie uma lista `<ul>` com 3 itens `<li>`.
2. Adicione um link `<a>` para [Google](https://www.google.com).
3. Insira uma imagem `<img>` (src: `https://via.placeholder.com/150`).
4. Crie um cabeçalho `<h1>` com o texto "Bem-vindo!".
5. Crie um parágrafo `<p>` com algum texto.
6. Crie um botão `<button>` com o texto "Enviar".
7. Crie um input `<input>` de texto com placeholder "Digite seu nome".
8. Crie um formulário `<form>` com input e botão.
9. Crie uma lista ordenada `<ol>` com 3 itens.
10. Crie a estrutura básica `<header>`, `<main>` e `<footer>`.
